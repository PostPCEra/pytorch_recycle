# pytorch_recycle


### Install
 - have Anaconda installed on ubantu, verify with $conda --version
 - install the following , which are reference in  .ipynb  jupyter notebook files
 - $conda install pytorch
 - $conda install matplotlib
 - $conda install jupyter
 - 
 - **RUN**
 - $jupyter start
